Lab 16: Fancy CSS 2

Challenge

The purpose of Lab 16 was to build on our understanding of what we learned about CSS previously through styling our Lab 16 webpage using more rules of CSS that we learned. We were to style the Lab 16 webpage using different methods of "fancy" CSS styling such as different backgrounds as well as transition, animation, and column elements. As a bonus, we were to add some of these effects to our homepage and create two other web pages that use these effects (both "tacky" and "classy"). After doing so we are to test our files locally to see if everything works the way it should and then upload them to the server.
