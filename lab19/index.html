Lab 19: CSS Frameworks

Challenge

The purpose of Lab 19 was to build on our understanding of what we learned about Bootstrap through styling our Lab 19 webpage using Bootstrap (layout, CSS and HTML). We were to add the Bootstrap dependencies to our HTMl, then incorporate Bootstrap's grid layout and add a navbar to the top of our page. As a bonus, we were to either implement a sticky navbar, a navbar with dropdowns, or add collapsible sections to our page. After doing so we are to test our files locally to see if everything works the way it should and then upload them to the server.

Problems

When completing Lab 19, the only issue that I had was getting my "Stuff" section to the right of the "Challenge", "Problems", and "Results" section. At first I had some difficulties, but eventually I found that I had to nest a row in a row in order to successfully put the sections next to each other.

Results

The results of this lab are shown on this page as it uses Bootstrap. It uses the grid layout and even has a navbar with links to previous labs that turns into a collapsible menu when the page is shortened.
