    The purpose of Lab 17 was to build on our understanding of what we learned about CSS previously through styling different html files using different elements of CSS layout styles that we learned. We were to style each page using different methods of layouts such as flexbox, grid, and float. We were to create a wireframe and model each page the same using the different elements. As a bonus, we were to use the grid element to style the layout as well. After doing so we are to test our files locally to see if everything works the way it should and then upload them to the server.

    Problems

    When completing Lab 17 I had no issues. I have used the float element previously and getting the hang of flexbox was very easy.
