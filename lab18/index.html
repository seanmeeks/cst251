    The purpose of Lab 18 was to build on our understanding of CSS layouts through styling our Lab 18 page using certain elements to create a responsive layout. We were to style the page using the meta tag and media query rule so that the viewport scales properly on mobile devices and the side-by-side elements shift to 100% width when the viewport is smaller than 600px. As a bonus, we were to style the webpage the same but without using the Media Query rules. After doing so we are to test our files locally to see if everything works the way it should and then upload them to the server.
